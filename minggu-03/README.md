# ARCHITECTURE CLOUD COMPUTING 

Everything as a Service

== Contoh Software as a Service adalah Dropbox.==
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/dropbox.jpeg)

Dropbox adalah layanan cloud yang berfungsi untuk menyimpan berbagai file. Cara menggunakan dropbox tidaklah sulit, kita bisa memakai layanan dropbox dengan gratis.

Setiap file yang terkoneksi akan otomatis terhubung dengan komputer, ponsel, atau device lainnya yang memakai aplikasi dropbox.

Dengan dropbox, kita bisa menyimpan file yang sudah kita buat dikomputer kantor tanpa harus membawanya melalui flashdisk ke komputer rumah atau sebaliknya.

Setiap file yang kita simpan di dropbox, otomatis file tersebut tersebar ke komputer/gadget yang memakai akun dropbox yang sama atau sudah tersinkronisasi. Syarat pertama untuk memulainya sobat harus memiliki akun dropbox terlebih dahulu.


== Cara Install Dropbox di PC atau Laptop ==

* Dari google chrome : Klik kanan file Dropbox yang sudah di download kemudian klik buka untuk memulai proses install
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/1.PNG)

* Jika user menggunakan Windows 7 maka akan muncul Pop up Security Warning, Klik Run untuk melanjutkan
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/2.PNG)

* Setup Wizard proses install dropbox akan mulai berjalan, Klik Install untuk melanjutkan
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/3.PNG)

* Proses instalasi Dropbox user akan berjalan

![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/4.PNG)

* Setelah proses instalasi selesai, pilih I already have a dropbox account dan dilanjutkan dengan klik tombol next
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/5.PNG)

* Masukkan alamat email user diatas dan password Dropbox sobat tadi serta beri nama Komputer user( tidak wajib ). Klik next untuk melanjutkan install-dropbox

![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/6.PNG)

* Pilih yang 2 GB ( Free ) kemudian Klik next

![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/7.PNG)

* Supaya user tidak bingung nanti pada saat selesai proses setup, Pilih menu Advanced (jangan yang Typical – recommended) kemudian klik next

![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/8.PNG)

* Letakkan Folder Dropbox di Local Disk C: ( Drive C: ) 

![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/9.PNG)

* Setelah user Klik OK maka akan terlihat seperti gambar dibawah ini ( C:\Dropbox ) kemudian Klik next lagi
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/10.PNG)

* Pilih opsi : I want this computer to sync all of the folder in my Dropbox kemudian dilanjutkan dengan klik install
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/11.PNG)

* Klik Skip Tour untuk melewati Tour

![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/12.PNG)

* Proses install Dropbox sudah selesai, Klik Finish untuk mengakhiri atau menutup wizard
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/13.PNG)

* Setelah user mengklik Finish maka folder Dropbox akan otomatis muncul ke hadapan sobat
![On Demand](https://github.com/Rohmatul1/tct/blob/master/minggu-03/images/14.PNG)

Selamat user, sudah mempunyai akun Dropbox dengan disk space 2 GB. Semoga artikel ini cukup untuk memberikan gambaran betapa mudahnya membuat akun Dropbox dan menginstallnya di PC atau Laptop user.
